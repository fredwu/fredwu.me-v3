So we want to dynamically instantiate an object (i.e. convert a string to a class constant).

Instead of doing this:

```ruby
@dynamic_var = "User"
eval(@dynamic_var)
```

You can do this:

```ruby
@dynamic_var = "User"
@dynamic_var.constantize
```
